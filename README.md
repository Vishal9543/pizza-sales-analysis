# 🍕 Pizza Sales Analysis Project

## 📌 Overview
This project analyzes pizza sales data using SQL, Excel, and Power BI to extract business insights.

## 🛠 Tools Used
- SQL (MySQL)
- Excel
- Power BI

## 📊 Key Insights
- Total Revenue Analysis
- Top & Bottom Selling Pizzas
- Sales by Category & Size
- Daily & Monthly Trends

## 📁 Project Structure
- data → Raw dataset
- sql → SQL queries
- dashboard → Power BI dashboard

## 🚀 Outcome
Helped identify customer behavior and sales patterns to improve business decisions.
